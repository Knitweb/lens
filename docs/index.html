Lens

A pure-Python interpret layer that reads the Knitweb fabric, focuses relevant chunks, and returns provenance-cited answers without requiring a live LLM or vector service.

View Repository Research Notes

Read Only

Lens consumes Pulse Web exports and source rows. It does not mutate fabric records or become a consensus layer.

Cited

Every answer keeps source ids, URIs, CIDs, node ids, and relation paths visible for downstream verification.

Offline First

The default RLM harness is deterministic and testable with python -m pytest. Live models are optional adapters.

Portable Context

Export ranked chunks as JSON context bundles, render them as Markdown, or replay them through /interpret.
